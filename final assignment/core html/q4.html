List of HTML Tags

  1. <html> - The root element that defines the start and end of an HTML document.
  2. <head> - Contains meta-information about the document, such as the title and links to stylesheets.
  3. <title> - Sets the title of the document, which appears in the browser's title bar or tab.
  4. <body> - Contains the content of the HTML document, such as text, images, links, etc.
  5. <p> - Defines a paragraph of text.
  6. <a> - Defines a hyperlink that links to another web page or location within the same page.
  7. <img> - Embeds an image in the HTML document.
  8. <ul> - Defines an unordered list, typically rendered as a bulleted list.
  9. <ol> - Defines an ordered list, typically rendered as a numbered list.
  10. <li> - Defines a list item, used within both unordered (`<ul>`) and ordered (`<ol>`) lists.
